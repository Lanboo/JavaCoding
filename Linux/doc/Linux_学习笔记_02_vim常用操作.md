# vim常用操作
> 为了查看日志方便，快速定位关键词。

``` shell
> vim xxx.log            // 打开某个文件
> G 或者 shift g         // 翻页至最后一页（光标定位到最后一行）
> ?key                   // 在当前文件中搜索key，并高亮显示
> ctrl b                 // 往回（向上）翻页
> ctrl f                 // 向前（向下）翻页
> :q                     // 退出
> :q!                    // 强制退出
```
